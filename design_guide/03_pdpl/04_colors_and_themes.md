# دليل التصميم: منصة PDPL - نظام الألوان والسمات

## 4. Color System & Theming

تعتبر منصة PDPL مثالاً متقدماً في تطبيق أنظمة التصميم المتغيرة (Theming)، حيث تقدم تجربة بصرية غنية ومختلفة تماماً بين الوضع الفاتح والداكن، مع الحفاظ على هوية احترافية وجذابة.

### فلسفة الألوان:

- **الوضع الفاتح:** نادر الاستخدام ويعتبر ثانوياً. يعتمد على خلفية رمادية فاتحة (`#f0f4f8`) مع لون أساسي كحلي داكن (`#273470`).
- **الوضع الداكن:** هو الوضع الافتراضي والأساسي للمنصة. يتميز بخلفية كحلية عميقة (`#0d1529`) مع لون تمييز تركواز ساطع (`#3db1ac`). هذا التباين العالي يخلق واجهة دراماتيكية وواضحة، مثالية لعرض البيانات والمؤشرات.

### متغيرات CSS الكاملة:

تم استخراج **523 متغير CSS**، مما يدل على نظام تصميم دقيق ومحكم للغاية. فيما يلي المتغيرات الأساسية التي تشكل هوية المنصة.

#### الوضع الفاتح (`:root`)

```css
:root {
  --radius: 0.75rem;
  --background: #f0f4f8;
  --foreground: #1c2833;
  --card: #fff;
  --card-foreground: #1c2833;
  --primary: #273470; /* كحلي */
  --primary-foreground: #fff;
  --secondary: #6459a7; /* بنفسجي */
  --accent: #3db1ac; /* تركواز */
  --destructive: #eb3d63; /* أحمر */
  --border: #3db1ac1f;
  --input: #3db1ac1f;
  --ring: #273470;
}
```

#### الوضع الداكن (`.dark`)

```css
.dark {
  --background: #0d1529;
  --foreground: #e1def5;
  --card: #1a2550;
  --card-foreground: #e1def5;
  --primary: #3db1ac; /* تركواز */
  --primary-foreground: #0d1529;
  --secondary: #6459a7; /* بنفسجي */
  --accent: #3db1ac;
  --destructive: #eb3d63;
  --border: #3db1ac1f;
  --input: #3db1ac1f;
  --ring: #3db1ac;
}
```

### تأثيرات الزجاج والتوهج (Glass & Glow):

تستخدم المنصة متغيرات مخصصة لتأثيرات الزجاج والتوهج، مما يضيف طبقات من العمق والجمال على الواجهة.

```css
/* متغيرات تأثير الزجاج */
:root {
  --glass-bg: #ffffffbf;
  --glass-border: #2734701a;
  --glass-shadow: 0 4px 24px #2734700f;
  --glass-blur: blur(20px);
}
.dark {
  --glass-bg: #27347040;
  --glass-border: #3db1ac1f;
  --glass-shadow: 0 8px 32px #0000004d;
}

/* متغيرات تأثير التوهج */
:root {
  --glow-primary: #27347026;
  --glow-accent: #3db1ac26;
}
.dark {
  --glow-primary: #3db1ac66;
  --glow-accent: #3db1ac4d;
}
```

يتم تطبيق هذه المتغيرات على البطاقات (`.glass-card`) والأشرطة الجانبية (`.glass-sidebar`) وعناصر أخرى لإعطائها مظهراً زجاجياً شبه شفاف، بالإضافة إلى إضافة توهج ناعم حول الأزرار والعناصر التفاعلية (`.glow-teal`, `.glow-purple`) لتعزيز جاذبيتها.
