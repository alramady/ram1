# دليل التصميم العملي: منصة PDPL (pdpl.rasid.vip)

هذا الدليل هو توثيق تقني وعملي شامل لجميع الجوانب البصرية والبرمجية لمنصة رصد تسريبات البيانات الشخصية (PDPL). تم استخراج جميع الأكواد والمؤثرات والتفاصيل مباشرة من الكود المصدري للمنصة لضمان الدقة الكاملة.

## محتويات الدليل:

1.  **المؤثرات البصرية والرسوم المتحركة (`01_visual_effects.md`)**
    - توثيق كامل لـ 35 حركة `@keyframes` فريدة، بما في ذلك حركات مميزة مثل `aurora-shift` و `scan-line`.

2.  **مكونات Framer Motion (`02_framer_motion.md`)**
    - تحليل 97 نمط حركة فريد، مع التركيز على الحركات المتدرجة (`staggerChildren`) والفيزياء الربيعية (`spring`).

3.  **الرسوم البيانية (`03_charts.md`)**
    - توثيق لأنواع الرسوم البيانية من `Recharts`، مع التركيز على التصميم الزجاجي (Glassmorphism) والألوان الديناميكية.

4.  **نظام الألوان والسمات (`04_colors_and_themes.md`)**
    - شرح كامل لنظام الألوان للوضعين الفاتح والداكن (الافتراضي)، مع استعراض 523 متغير CSS وتأثيرات التوهج.

5.  **النصوص العربية (`05_arabic_strings.md`)**
    - قائمة شاملة بجميع النصوص العربية المستخدمة في الواجهة (1,084 نص فريد).
