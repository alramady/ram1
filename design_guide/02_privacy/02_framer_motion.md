# دليل التصميم: منصة الخصوصية - مكونات Framer Motion

## 2. Framer Motion Animations

تعتمد منصة الخصوصية بشكل كبير على مكتبة `Framer Motion` لإضافة حركات تفاعلية وسلسة تعزز تجربة المستخدم. تم استخراج **109 مؤثرات حركة فريدة** تطبق على المكونات المختلفة.

### أنماط الحركة الرئيسية:

- **دخول المكونات (Entrance):** معظم المكونات والصفحات تستخدم حركة ظهور تدريجي مع انزلاق خفيف للأعلى (`opacity: 0, y: 20` إلى `opacity: 1, y: 0`). هذا يخلق إحساساً بالترتيب والتدرج في عرض المحتوى.
- **التفاعل عند المرور (Hover):** يتم تكبير العناصر قليلاً عند مرور الفأرة فوقها (`whileHover: { scale: 1.05 }`) لإعطاء رد فعل فوري للمستخدم.
- **التفاعل عند النقر (Tap):** يتم تصغير العناصر قليلاً عند النقر (`whileTap: { scale: 0.98 }`) لتأكيد الاستجابة للنقر.
- **تأثيرات القوائم (Stagger):** يتم عرض عناصر القوائم بتأخير بسيط متتابع (`staggerChildren`) مما يخلق تأثيراً موجياً جذاباً.

### أمثلة على كود الحركة (Variants):

```javascript
// حركة الدخول القياسية للصفحات والبطاقات
const pageVariants = {
  initial: { opacity: 0, y: 20 },
  animate: { opacity: 1, y: 0 },
  transition: { duration: 0.5, ease: [0.4, 0, 0.2, 1] }
};

// حركة ظهور القوائم المنسدلة
const dropdownVariants = {
  initial: { opacity: 0, scale: 0.95, y: -10 },
  animate: { opacity: 1, scale: 1, y: 0 },
  exit: { opacity: 0, scale: 0.95, y: -10 },
  transition: { duration: 0.2, ease: "easeOut" }
};

// حركة تفاعل الأزرار والبطاقات
const interactiveVariants = {
  whileHover: { scale: 1.03, transition: { type: "spring", stiffness: 300 } },
  whileTap: { scale: 0.97 }
};

// حركة تدرج ظهور عناصر القائمة
const listVariants = {
  animate: {
    transition: { staggerChildren: 0.08 }
  }
};

const itemVariants = {
  initial: { opacity: 0, y: 20 },
  animate: { opacity: 1, y: 0 }
};
```

### تحليل تفصيلي للحركات حسب المكون:

| المكون/الصفحة | الحركة المطبقة | الوصف |
| :--- | :--- | :--- |
| **الصفحات الرئيسية** | `initial: {opacity:0, y:20}, animate: {opacity:1, y:0}` | حركة دخول قياسية لجميع الصفحات لتوحيد التجربة. |
| **البطاقات (Cards)** | `whileHover: {scale:1.03, y:-5}` | ترتفع البطاقة قليلاً عند المرور فوقها لجذب الانتباه. |
| **الأزرار (Buttons)** | `whileTap: {scale:0.95}` | استجابة مرئية فورية عند النقر. |
| **القوائم المنسدلة** | `initial/exit: {opacity:0, height:0}` | فتح وإغلاق سلس للقوائم. |
| **الأيقونات** | `whileHover: {scale:1.2, rotate: 10}` | حركة تفاعلية صغيرة وممتعة للأيقونات. |
| **الرسوم البيانية** | `transition: {staggerChildren: 0.1}` | ظهور أعمدة أو نقاط الرسم البياني بشكل متتابع. |
| **الجداول** | `initial: {opacity:0, x:-20}` | ظهور صفوف الجدول من اليسار لليمين. |

هذا الاستخدام المكثف والمدروس لـ `Framer Motion` يساهم بشكل كبير في جعل واجهة منصة الخصوصية تبدو حديثة وسريعة الاستجابة وتفاعلية للغاية.
