# دليل التصميم العملي: منصة الخصوصية (privacy.rasid.vip)

هذا الدليل هو توثيق تقني وعملي شامل لجميع الجوانب البصرية والبرمجية لمنصة الخصوصية. تم استخراج جميع الأكواد والمؤثرات والتفاصيل مباشرة من الكود المصدري للمنصة لضمان الدقة الكاملة.

## محتويات الدليل:

1.  **المؤثرات البصرية والرسوم المتحركة (`01_visual_effects.md`)**
    - توثيق كامل لـ 34 حركة `@keyframes` مع الكود المصدري.

2.  **مكونات Framer Motion (`02_framer_motion.md`)**
    - تحليل 109 مؤثر حركة فريد، مع أمثلة كود وتوزيعها على المكونات.

3.  **الرسوم البيانية (`03_charts.md`)**
    - توثيق لأنواع الرسوم البيانية المستخدمة من مكتبة `Recharts` مع نظام الألوان وأمثلة الكود.

4.  **نظام الألوان والسمات (`04_colors_and_themes.md`)**
    - شرح كامل لنظام الألوان للوضعين الفاتح والداكن مع متغيرات CSS الكاملة وتأثير الزجاج.

5.  **النصوص العربية (`05_arabic_strings.md`)**
    - قائمة شاملة بجميع النصوص العربية المستخدمة في الواجهة (2,574 نص فريد).
