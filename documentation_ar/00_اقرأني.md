# التوثيق التقني الشامل — منصة راصد الوطنية

> **الإصدار:** 1.0.0 | **التاريخ:** 13 فبراير 2026 | **النطاق:** [www.rasid.vip](https://www.rasid.vip)

تم إنشاء هذا المستودع من خلال **الهندسة العكسية الشاملة** لمنصة راصد الوطنية (منصة راصد الوطنية). يحتوي على 22 مستندًا تقنيًا عبر 9 أقسام، تغطي كل جانب من جوانب المنصة بدءًا من الرؤية الاستراتيجية وصولاً إلى تحليل الكود على مستوى الملف.

## نظرة عامة على المنصة

**منصة راصد الوطنية** هي منصة إدارة محتوى شاملة مصممة لتقديم الأخبار ومقاطع الفيديو والصور والأدلة والعروض التقديمية والروابط لجمهور عربي. تتميز بنظام تحكم في الوصول المستند إلى الأدوار مع 7 أدوار مستخدم، وسير عمل للموافقة على المحتوى، ونشر مجدول، وتحليلات شاملة.

## المكدس التقني

| الطبقة | التقنية |
|:---|:---|
| **الواجهة الأمامية** | React 19 + TypeScript + Vite 6 + TailwindCSS |
| **مكتبة واجهة المستخدم** | shadcn/ui (Radix UI) + Framer Motion |
| **واجهة برمجة التطبيقات** | tRPC 11 (22 موجهًا، 95 إجراءً) |
| **قاعدة البيانات** | MySQL/TiDB + Drizzle ORM |
| **المصادقة** | JWT + ملفات تعريف الارتباط HTTP-only |
| **تخزين الملفات** | Manus CDN |
| **النشر** | منصة Manus |
