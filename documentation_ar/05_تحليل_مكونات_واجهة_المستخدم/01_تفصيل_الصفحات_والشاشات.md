# 05: تحليل مكونات واجهة المستخدم

يقدم هذا القسم تحليلاً مفصلاً لمكونات واجهة المستخدم المستخدمة في منصة راصد الوطنية، بما في ذلك المكونات المخصصة ومكونات المكتبات الخارجية. يغطي التحليل وظائفها وقابليتها لإعادة الاستخدام والتزامها بنظام التصميم العام.

## 1. مكونات واجهة المستخدم الأساسية (shadcn/ui & Radix UI)

يستفيد التطبيق من مكتبة مكونات `shadcn/ui`، المبنية على Radix UI، لتوفير مجموعة أساسية من بدائيات واجهة المستخدم التي يمكن الوصول إليها وغير المصممة. يضمن هذا النهج تجربة مستخدم متسقة وعالية الجودة مع السماح بتخصيص واسع النطاق ليتناسب مع العلامة التجارية للمنصة.

| ملف المكون | الوصف |
| :--- | :--- |
| `avatar.tsx` | مكون واجهة المستخدم للصورة الرمزية (Radix) |
| `badge.tsx` | مكون واجهة المستخدم للشارة |
| `button.tsx` | مكون واجهة المستخدم للزر |
| `card.tsx` | مكون واجهة المستخدم للبطاقة |
| `checkbox.tsx` | مكون واجهة المستخدم لمربع الاختيار |
| `dialog.tsx` | مكون واجهة المستخدم للحوار/النافذة المشروطة (Radix) |
| `dropdown-menu.tsx` | مكون واجهة المستخدم للقائمة المنسدلة |
| `input.tsx` | مكون واجهة المستخدم للإدخال |
| `label.tsx` | مكون واجهة المستخدم للتسمية |
| `popover.tsx` | مكون واجهة المستخدم للنافذة المنبثقة |
| `scroll-area.tsx` | مكون واجهة المستخدم لمنطقة التمرير |
| `select.tsx` | مكون واجهة المستخدم للاختيار |
| `separator.tsx` | مكون واجهة المستخدم للفاصل |
| `sheet.tsx` | مكون واجهة المستخدم للورقة/الدرج |
| `sonner.tsx` | إشعار منبثق (Sonner) |
| `switch.tsx` | مكون واجهة المستخدم لمفتاح التبديل |
| `tabs.tsx` | مكون واجهة المستخدم لعلامات التبويب |
| `textarea.tsx` | مكون واجهة المستخدم لمنطقة النص |
| `tooltip.tsx` | مكون واجهة المستخدم للتلميح |

## 2. مكونات التطبيق المخصصة

بالإضافة إلى المكتبة الأساسية، تتميز المنصة بمجموعة من المكونات المخصصة المصممة لتلبية احتياجات التطبيق المحددة. تغلف هذه المكونات وظائف معقدة وهي مصممة لإعادة الاستخدام عبر صفحات ووحدات مختلفة.

| ملف المكون | الوصف |
| :--- | :--- |
| `AnimatedCounter.tsx` | يعرض رقمًا يتحرك من قيمة بداية إلى قيمة مستهدفة. |
| `BookmarkButton.tsx` | زر ذو حالة يسمح للمستخدمين بوضع إشارة مرجعية على المحتوى. |
| `CommentsSection.tsx` | مكون شامل لعرض وتقديم التعليقات. |
| `ErrorBoundary.tsx` | مكون React لالتقاط أخطاء JavaScript في أي مكان في شجرة المكونات الفرعية الخاصة به. |
| `FloatingParticles.tsx` | يعرض رسومًا متحركة للجسيمات العائمة جذابة بصريًا للخلفيات. |
| `Footer.tsx` | تذييل التطبيق القياسي، الذي يحتوي على روابط ومعلومات حقوق النشر. |
| `GuidedTour.tsx` | يوفر جولة إرشادية خطوة بخطوة للمستخدمين الجدد لتعلم ميزات التطبيق. |
| `Layout.tsx` | هيكل التخطيط الرئيسي لجميع الصفحات، وعادة ما يتضمن شريط التنقل والتذييل. |
| `LikeButton.tsx` | زر للمستخدمين للإعجاب أو عدم الإعجاب بقطعة من المحتوى. |
| `MediaViewer.tsx` | نافذة مشروطة أو صندوق ضوئي لعرض الصور وأنواع الوسائط الأخرى. |
| `Navbar.tsx` | شريط التنقل الأساسي للتطبيق. |
| `NotificationBell.tsx` | أيقونة تعرض عدد الإشعارات غير المقروءة وتفتح لوحة الإشعارات. |
| `PageTransition.tsx` | مكون يضيف انتقالات متحركة عند التنقل بين الصفحات. |
| `PdfReportButton.tsx` | زر يؤدي إلى إنشاء وتنزيل تقرير PDF. |
| `PresentationMode.tsx` | يتيح وضع عرض تقديمي بملء الشاشة وخالٍ من الإلهاء للمحتوى. |
| `PresentationToolbar.tsx` | شريط أدوات به عناصر تحكم للتنقل والتفاعل مع العروض التقديمية. |
| `RealtimeNotifications.tsx` | نظام لتلقي وعرض الإشعارات في الوقت الفعلي. |
| `RichTextEditor.tsx` | محرر نصوص منسقة يعتمد على Tiptap لإنشاء وتحرير المحتوى المنسق. |
| `ScrollProgressBar.tsx` | شريط تقدم يشير إلى موضع تمرير المستخدم على الصفحة. |
| `SectionHeader.tsx` | رأس موحد لأقسام مختلفة من الصفحة. |

## 3. مكونات الرسوم المتحركة والحركة

تتضمن المنصة العديد من المكونات المخصصة لإنشاء تجربة مستخدم ديناميكية وجذابة من خلال الرسوم المتحركة والتفاعلات الدقيقة. تم بناء هذه بشكل أساسي باستخدام Framer Motion.

| ملف المكون | الوصف |
| :--- | :--- |
| `animations/AnimatedCounter.tsx` | يوفر منطق الرسوم المتحركة لمكون `AnimatedCounter`. |
| `animations/KineticText.tsx` | ينشئ تأثيرات طباعة حركية للنص. |
| `animations/MicroInteractions.tsx` | مجموعة من الرسوم المتحركة الصغيرة والدقيقة لتفاعلات المستخدم. |
| `animations/ParticleBackground.tsx` | تنفيذ تأثير خلفية الجسيمات. |
| `animations/ScrollReveal.tsx` | يحرك العناصر إلى العرض أثناء قيام المستخدم بالتمرير لأسفل الصفحة. |
